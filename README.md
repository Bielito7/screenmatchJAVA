# Screenmatch Sistema de Filmes e Séries 🎬
 MVP (Minimum Viable Product)

Projeto desenvolvido para praticar os conceitos de Programação Orientada a Objetos (POO) em Java.

## Conceitos aplicados

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Interfaces
- Getters e Setters

## Sobre o projeto

O sistema simula o backend de uma plataforma de filmes e séries, permitindo aplicar na prática os principais pilares da orientação a objetos e compreender como esses conceitos são utilizados em aplicações reais.

## Objetivo

O foco deste projeto foi fortalecer meus conhecimentos em Java e desenvolver uma melhor compreensão da estruturação de sistemas utilizando boas práticas de Programação Orientada a Objetos.
